# Rain Today Weather App
[![NPM](https://img.shields.io/npm/l/angular)](https://github.com/d4njunior/rain-today/blob/main/LICENSE) 

## About

A weather application to get the temperature, weather condition and show a custom image if it is day or night based on the weather climate forecast. The app uses an API to search the particular city by the IP address of the user. Powered by <a href="https://www.weatherapi.com/" title="Free Weather API">WeatherAPI.com</a>

## Run

```bash
# Clone repository
git clone https://github.com/d4njunior/rain-today

# To setup the dependencies run
npm i

# To setup the environment API key
got to www.weatherapi.com to get one and change the environment file

# To run the app
ng serve --open
```
